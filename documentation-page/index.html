Introduction

JavaScript is the world's most popular programming language. JavaScript is the programming language of the Web and easy to learn. It is a lightweight compiled programming language with first-class functions.

JavaScript is one of the 3 languages all web developers must learn:

  • HTML to define the content of web pages.
  • CSS to specify the layout of web pages.
  • JavaScript to program the behavior of web pages.
Hello world
You can start by opening Scratchpad and write your first "Hello world" JavaScript code: function greetMe(yourName) { alert("Hello " + yourName); } greetMe("World"); Select your code in the pad and press Ctrl+R to view in your browser.
JavaScript used in HTML

JavaScript code is inserted between <script> and </script> tags in HTML:

<script> document.getElementById("demo").innerHTML = "My First JavaScript"; </script>

Scripts can be placed in the <body> and/or in the <head> section of an HTML page.

For example, in the <head>:

<head> <script> function myFunction() {document.getElementById("demo").innerHTML = "Paragraph changed."; } </script> </head>

While in the <body>, it would look like this:

<body> <script> function myFunction() {document.getElementById("demo").innerHTML = "Paragraph changed."; } </script> </body>

Scripts can also be placed in external files. JS files have the file extensions .js.You can include an external JS file in your HTML page by placing the name of the file in the source attribute like this:

<script src="myScript.js"></script>
JavaScript Output

You can display data in JavaScript in differnt ways:

  • Using innerHTML into an HTML element.

  • Using document.write() into the HTML output.

  • Using window.alert() into an alert box.

  • Using console.log() into the browser console.
